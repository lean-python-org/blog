The When of Python Isn't Dead, It's Resting!

  • Mon 23 January 2023
  • Python
  • Grant Paton-Simpson
  • #python

The When of Python project has a core vision - making it easier for people to learn and use Python by clarifying when features should be used and when not.

As a start, we have presented a conference talk, done some initial research on feature usage in the wild, refined the concept in response to feedback, and written some initial recommendations. The next steps are:

  1. Make a reference site providing succinct guidance on when to use specific features (and when not to).
  2. Find a suitable PEP-Delegate to help get the When Of Python added as a PEP (PEP 21?). See PEP Purpose and Guidelines
  3. Miscellaneous items on our Trello board.

For work reasons we are pausing the project ready to pick up again at a later point.

Python still needs a When of Python and even though work on steps 1-3 is paused we still welcome feedback on the project.
